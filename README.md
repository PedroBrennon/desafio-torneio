Desafio - Torneio com Chaveamento

Pegando um gancho na copa do mundo, monte o sistema onde os torneios são tipicamente mata-mata são compostos por times e chaves, onde são gerados os vencedores de cada chave até chegar ao grande vencedor do torneio. 

Seu objetivo é criar um sistema onde é possível cadastrar os times e gerenciar o torneio até a fase final, onde poderemos eleger o vencedor.

Solução

O sistema foi feito em ASP.NET C#, utilizando API e Web, localDB em SQL Server e storage procedures.

No arquivo contém uma imagem onde foi baseado o projeto todo, através do chaveamento dos times, seguindo a ordem de oitavas, quartas, semifinal e final.
